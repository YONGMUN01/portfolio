틈새빛 웹 개발 프로젝트

JSP 기반 웹 사이트를 직접 설계·구현하며
사이트 서비스를 구축한 프로젝트

JAVA Oracle MyBatis Apache Tomcat HTML JavaScript CSS JDK Servlet JSP JSTL

프로젝트 개요

쉬었음 청년들의 경제활동 감소로 인한 경제적·사회적 이슈로 인한 문제를 해결하기 위해 정기적인 모임 및 수업 활동을 통해 청년들의 경제적·사회적 활동을 증가시키기 위한 웹 사이트 제작 프로젝트입니다.

진행 과정

  • HTML, CSS, JavaScript를 활용한 웹 페이지 제작
  • Eclipse환경에서 JAVA를 사용해 JSP 사이트 제작
  • MVC2 패턴을 기반으로 설계
  • Oracle DB를 이용한 DB 구축 및 MyBatis를 통한 연동
  • 동기 및 비동기 요청을 통해 데이터 처리

문제와 해결

초기 ERD 설계 및 정규화 과정에서 과도한 정규화를 적용하여 유지보수가 어려웠으며 데이터를 처리하는 과정에서 과도한 처리가 이루어졌습니다.
이를 해결하기 위해 정규화를 완화해 구조를 단순화하였고, 참조를 줄여 데이터 처리 과정을 줄였습니다.

결과 및 성과

  • 사이트 이용 시 지연시간 감소
  • 웹 페이지 간 정상적인 이동 및 데이터 정상 처리
  • 웹 사이트의 구조 확인 및 보안적 안목 증가
